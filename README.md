# Cultural Aspects

Windows child computer-locking and parent reporting project.

## Current Stage

Step 6: Admin Startup Setup

The project currently contains a Python/Tkinter Windows lock-app prototype, local reporting data, and administrator startup setup scripts. The mobile reporting app is not implemented yet.

## Main Reference Files

- `PROJECT_REFERENCE.md` contains the stable project baseline and recovery notes.
- `AGENTS.md` contains the project agents and their responsibilities.
- `docs\DEVELOPMENT_STEPS.md` tracks the approved step-by-step development plan.

## Folder Overview

- `src\windows-lock-app` - Windows locking application prototype.
- `src\mobile-reporting-app` - future mobile parent reporting app.
- `config` - example configuration files only.
- `data` - local runtime data location, not committed.
- `logs` - runtime logs location, not committed.
- `scripts\admin` - administrator setup scripts.
- `docs\ADMIN_STARTUP_SETUP.md` - administrator startup task setup instructions.
- `docs` - project documentation.
